# Global Electronics Retail Analysis 📊

An end-to-end data analytics project using **Python, SQL, and Power BI** to optimize retail performance.

## 🚀 Key Features
- **Data Pipeline:** Cleaned and merged 5+ datasets using Python.
- **SQL Analysis:** Solved 10 complex business problems including Pareto 80/20 profit analysis and CLV.
- **Power BI:** Interactive dashboard for executive decision-making.

## 📈 Business Insights
- Identified that 80% of profit comes from just 2 product categories.
- Discovered seasonal peaks in Computer sales during August and December.
- Calculated Avg CLV per country to identify high-value markets.

## 🛠 Tech Stack
- **Python:** Pandas, NumPy
- **SQL:** MySQL (CTEs, Window Functions)
- **Visualization:** Power BI, Excel
